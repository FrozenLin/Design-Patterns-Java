

# Brief overview of GoF Design Patterns

The GoF described 23 patterns.

#### A. Creational Patterns

- Singleton Pattern
- Prototype Pattern
- Factory Method Pattern
- Builder Pattern
- Abstract Factory Pattern

#### B. Structural Patterns

- Proxy Pattern
- Flyweight Pattern
- Composite Pattern
- Bridge Pattern
- Facade Pattern
- Decorator Pattern
- Adapter Pattern

#### C. Behavioral Patterns

- Observer Pattern
- Strategy Pattern
- Template Method Pattern
- Command Pattern
- Iterator Pattern
- Memento Pattern
- State Pattern
- Mediator Pattern
- Chain of Responsibility Pattern
- Visitor Pattern
- Interpreter Pattern
